Gamma097’s Custom Steam Grid Icon Pack

How to use these icons:

1.            Extract the files in the RAR to a place you can easily find.

2.            Open up Steam and switch to the Library tab.

3.            Either click on the “Grid View icon to the right of the zoom slider (It may be somewhere else if you have a custom skin in place) or click on View, Games Grid.

4.            Right click on the game/tool in question.

5.            Click on Set Custom Image...

6.            A dialog box should show up.  Click on Browse...

7.            A Windows Explorer window will show up.  Find the icon corresponding to the game/tool.  Click it and click OK.  Click on Set Image in the first dialog box.

8.            If you decide you don't like the icon you have set, right click on the icon in question and click on Remove Custom Image...

9.            On the new dialog box, click Continue.  The icon should revert to Steam’s default icon.

 

If you have a request, I'll do my best to make it happen.  If there's an inaccuracy in the file name of an icon, politely say so, and I'll fix it for the next version.
